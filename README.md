# SimpleChat v0.1 ヽ( ・∀・)ノ
> by Mateusz Ożóg

Simple NodeJS API based chat!

## Requirements

- NodeJS

## Installation

Simple Installation of this simple app!

```
$ npm install
```

## Usage

One simple command!

```
$ npm run chat
```

Chat server will start on port :3000 on your local machine\n
Clients can join just going to your IP/WWW adress :3000!
